# このプロジェクト（実験）の概要
スペックの余っているデスクトップPCで、棋棋譜解析を行わせる個人開発システムの実証実験をしたいです。
基本的に TypeScript フルスタック開発をイメージしており、pnpm monorepo をセットアップしてもらいたいです。
パッケージ名は @daiius/shogi{,-server,-worker} のようなイメージです。

システム全体の構成はこんなイメージです。

- 棋譜保存操作と将棋盤表示を行う web インターフェース
  - クリップボードからのkif形式ファイル貼り付け
- 棋譜データを記録するデータベース（Drizzle ORM + MySQL）と API サーバー（Hono）
  - API サーバーからのデータ取得は API\_KEY が必要 
- デスクトップPCで動作するワーカー（サーバーの棋譜データを一定時間間隔で取得し、yaneura-ou + メジャーな評価関数エンジンを呼んで棋譜解析を行う）
  - API\_KEY を使ってサーバーにアクセスする
